Airbnb activity

Last week, I finished an activity called “Airbnb”, focusing on gathering and arranging data and plotting graphs of house distribution in a region. After that, I want to elaborate on some of my ideas.

First time I found its intrinsic beauty when I putted data from a document into Python and started to edited them. Moreover, I can plot a graph about these information using several line of codes, making data visualized into graph. I think all the processes are logical because they are controlled by computer calculating system.

In addition, I understand the myriad and diverse applications of mathematics in areas. I think mathematics here can be separated into two aspects. First is mathematics knowledge like number, arithmetic, which is reading data and integrating numbers. Second is logical ability, writing code needs to pay attention to the sequence and logic, otherwise it is easy to make mistakes.

I think I gain diverse range of skills from learning mathematics as mathematics represents logic so mathematics really helps me a lot. Moreover, good determination is quite essential, I was nervous when I first learned about this project because it was a whole new area for me. However, through asking questions at class and discussed with classmates, I finally overcome the difficulties so that I think I can insist on doing something, also, I don’t give up easily before success. In addition, time management is necessary as recovering and doing homework are parts of my job so I need to plan everything on a list to be organized.
