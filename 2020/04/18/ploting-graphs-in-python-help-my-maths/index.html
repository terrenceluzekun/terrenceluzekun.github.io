functions in python help my maths

As I mentioned in past diary, I began to study python few months ago. After some contact in python, I found there are really lots of functions not only help programmers to make editing codes much easier, but also help people in daily life. For instance, I have studied how to draw graphs on python through codes. After the lesson, I wondered whether this function could draw some mathematics’ graphs and help me understand questions in Maths more easily. Finally, I found python can do it !

Here are some brief explanations,

Firstly, in order to decrease time for coding, I shrink the language “numpy” and “matplotlib” into “np” and “plt” seperately by importing them. Next, “plt.figure()” is used to plot a diagram with basic axises as a background.

Secondly, as I want two diagrams appearing together, I marked each diagram as a subplot. Then, here are some explanations of (2,1,1) and (2,1,2), the number on left represents number of rows, there are two diagrams and they are on two lines. Middle number represents number of column while there is only one column. Right one is the identifier of graph so “1” for first graph and “2” for the rest one obviously.

Finally, we just need to write codes in the form of “plt.subplot” while sin(x) and cos(x) are functions in “numpy” language so we use “np.()” to derive these two functions.
